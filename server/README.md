# Server (Vite + Express)

## Scripts

```bash
npm run dev
npm run build
npm run start
```

- `dev` runs the server with `vite-node --watch`.
- `build` outputs a production bundle in `dist/server.js`.
- `start` runs the bundled server.

## Environment

Create `server/.env` from `server/.env_example` and configure database and port values.
